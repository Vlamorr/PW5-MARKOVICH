Question №1

a. What is a box model in CSS. Give box-sizing property definition?

The box model in CSS defines how elements are laid out on a webpage, including their content, padding, border, and margin. The box-sizing property defines how these dimensions are calculated, with options like content-box and border-box.

Question №2

b. What is a “clearfix hack” and when does it need to be applied?

The "clearfix hack" is a technique used to clear floated elements within a container. It's applied when a container doesn't fully contain its floated child elements, causing layout issues.

Question №3

c. What is a “margin collapse” and what are the use cases for margin: auto and negative margin values?

Margin collapse occurs when the vertical margins of adjacent elements overlap, collapsing into a single margin. margin: auto is used to center an element horizontally within its container, while negative margin values can be used for adjustments or creating overlapping effects.

Question №4

d. What was the problem with achieving Holy Grail layout?

The problem with achieving the Holy Grail layout was creating a webpage layout with a header, footer, and three columns, where the center column was the tallest and the side columns were of equal height, all while maintaining a fluid and responsive design.

Question №5

e. What are the modern ways to achieve Holy Grail layout?

Modern ways to achieve the Holy Grail layout include using flexbox or CSS Grid, which offer more robust and flexible solutions compared to older techniques like floats and positioning. These newer methods provide better control over layout and responsiveness.
